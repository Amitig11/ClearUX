What we implemented

This working MVP checks one pattern: Exit-Flow Asymmetry. This means the website makes cancellation harder than subscribing or staying subscribed.

To avoid vague testing, we used exact measurable values: number of steps, button position, text size, button area, contrast ratio, and confidence score.

Strong warningRatio ≥ 2:1 and confidence ≥ 80%
Medium warningConfidence 50%–79%
No warningNo subscription/cancellation flow found
